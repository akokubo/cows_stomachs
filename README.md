## ゲームの概要(description)

べこっこ(ウシ)をおいしく育てて出荷！

ウシ(cow)は胃(stomachs)が4つあるんです！
4つの胃を繰り返し(continute)使って、反芻(はんすう: rumination)します！！
それをゲームにしました！！

1. まず、ウシに、エサを与えます。
2. 次に、エサを消化する胃の順番を選びます。  
  胃には消化特性があり、エサに対して最適な順番があります。  
  4つの胃で消化を繰り返す(continue)ことでウシが育ちます。
3. ウシがやせたり、太りすぎないよう、ほどよくおいしく育てましょう。
